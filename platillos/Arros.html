Ingredientes para preparar Arroz Rojo:

Sigue la receta para preparar Arroz Rojo

Ingredientes:

    • pasos de preparacion:


    1.El arroz se remoja 15 minutos en agua caliente y se escurre perfectamente.
    2.Las zanahorias se limpian y se cortan en cubitos pequeños;
    3.el jitomate se licua con ajo y la cebolla con muy poca agua.
    4.En una cacerola se pone suficiente para freir el arroz hasta que tome un color dorado medio, se le retira el exceso de aceite,
    5.se le agregan las zanagorias,el jitomate licuado y colado, un poco de consome o sal al gusto para que se fria el jitomate, no debe moverse bruscamente para que no se bata.
    6.Se le añade el agua o caldo de pollo y la rama de perejil.
    6.Cuando empieza a hervir se baja el fuego y se deja cocer aproximadamente 1/2hora.
